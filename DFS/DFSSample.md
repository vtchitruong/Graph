Find a path from one node to another using DFS.  
Return -1 if no path found.  

### Constraints
0 < n < 10  (number of vertices/nodes)  

### Sample Input
8 7 1 5  
1 2  
1 3  
2 3  
2 4  
3 5  
4 6  
7 8  

### Sample Output
1 2 3 5  

### Explanation
8 node  
7 edge  
Find a path from node 1 to node 5.  
The path is 1 2 3 5.
